# Hedamo Disclosure System

A modern, high-performance web application designed to display producer-declared information for various products. Built with Next.js, React, and Tailwind CSS.

- **Sorting**: Sort products by name (ascending/descending) or date (latest/oldest).
- **Interactive UI**: Smooth animations powered by Framer Motion.

## üì¶ Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Krigoy/Disclosure-System.git
   cd Disclosure-System
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üìÅ Project Structure
